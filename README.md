## 📌 Project Overview
Hotel executives often need a centralized view of operational performance to monitor revenue, occupancy, reservation trends, and customer behavior across multiple properties. The goal of this project was to design an executive-level Power BI dashboard that transforms raw hotel booking data into actionable insights that support strategic decision-making.

Unlike traditional reports that simply display metrics, this dashboard is organized around business questions, allowing stakeholders to quickly understand performance and identify areas that require attention.

# 🎯 Business Objective

The objective of this project was to build an executive dashboard capable of helping leadership:

- Monitor overall hotel performance
- Track revenue generation
- Evaluate occupancy trends
- Analyze reservation demand
- Monitor cancellation rates
- Compare hotel performance
- Understand customer booking behavior
- Explore data through interactive filtering

# ❓Business Questions Answered

This dashboard was designed to answer key business questions including:

- How much revenue have our hotels generated?
- What percentage of available rooms are currently occupied?
- How many reservations have been made?
- What is the current cancellation rate?
- How did booking demand trend throughout the year?
- How are reservations progressing through the booking pipeline?
- Which markets generate the highest reservation demand?
- Which hotel properties contribute the most revenue?
- Which customer segments book with us most frequently?

# 🎨 Design Approach

One of the primary goals of this project was to improve executive storytelling through dashboard design.

Instead of using traditional chart titles such as:

- Revenue by Hotel
- Bookings by Country
- Bookings by Month

the dashboard was intentionally organized around business questions such as:

- How did booking demand trend throughout the year?
- Which markets are driving the highest reservation demand?
- Which properties contribute most to total revenue?

This storytelling approach enables executives to focus on answering business questions rather than interpreting technical reports.
